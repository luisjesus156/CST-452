Cloud Test Application for Product Management

This cloud-based web application enables users to securely register, log in, and manage events. It demonstrates the use of secure coding practices, input sanitization, and component-based design.

Key Features

  • User registration and login with encrypted credentials
  • Event management: create, list, and search events
  • Security: SQL Injection and XSS protection
  • Technologies: Spring Boot, Thymeleaf, MySQL (MAMP), MVC architecture

Purpose

The project highlights the importance of building secure, scalable, cloud-based applications. It aligns with real-world standards for product management platforms and cloud deployment environments.
